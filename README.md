# react-rxjs

from [fdecampredon/react-rxjs-todomvc](https://github.com/fdecampredon/react-rxjs-todomvc)

using [Milkcocoa](https://mlkcca.com/)

## Install Package

```
npm i
```


## Build

```
npm run install
```

## Start

```
node server.js
```

and open `http://localhost:3000` in your browser.
